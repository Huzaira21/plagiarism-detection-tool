# Plagiarism Detection Tool

A feature-rich plagiarism detection system built in C++ 
using core Data Structures concepts.
Developed at Namal University, Mianwali.

## Features

- **Sentence-Level Plagiarism** — Exact sentence matching
- **Word-Level Plagiarism** — Compares important words 
  after removing stop words
- **Synonym Detection** — Detects copied content using 
  synonyms from `synonyms.txt`
- **Approximate Sentence Matching** — Detects modified 
  sentences using Levenshtein Distance
- **Cosine Similarity** — Measures document similarity 
  using word frequencies
- **N-Gram Detection** — Compares n consecutive words 
  (bigram, trigram, any n)

## How to Run
> **Note:** Uses Windows-specific libraries
> Open `main.cpp` in Dev C++ and press F11
